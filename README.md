# Roberthood

Users can read up on latest business news, check stock quotes, add ticker to watchlist, and buy and sell stocks in a portfolio.

[Live Site](https://roberthood-trading-platform.herokuapp.com/)

## Technologies Used

* React JS
* Ruby on Rails
* IEX API
* News API
* Recharts API
* Ajax
* HTML3
* CSS5
* Firebase Database
